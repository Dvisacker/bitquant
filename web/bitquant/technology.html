Bitquant uses a home grown open source technology platform which is available on our github site.

Technology we use

Some of the components we have used are:
  • Docker
  • Jupyter and Python
  • Mageia Linux.
  • Dokuwiki
  • Quantlib.
A bundled set of our platform is available in the image bitstation:latest from docker hub. The software which we write are mostly back office. For trading we use the standard interfaces.

Also for backend processing, we use Odoo

Technology we avoid

  • Java - Java is a big bureaucratic language designed for big bureaucratic companies.
  • Anything closed source - We can't tinker with the software

Technology we are looking for

This is stuff that we are looking for, and if we don't find it, we might as well write it ourselves. We are willing to pay for market data feeds and support, but the project must be open source.
  • Clean data feeds - Right now most cheap market data requires you to go through the UI of the market
  • Trade journal software - There is no easy open source software that lets you look over the past history
  • Chart visualization tools that aren't designed to screw over the trader
