This application is bittrader, a trading system based on pulling together open-source javascript applications into a single trading system. All of the code here is BSD licensed and is available via our Github repository with our Wiki

To check out our progress see the Bitquant blog

Our target date for alpha for an MVP (minimally viable product) is August 2014. The features for the initial release are

  • 100% open source (BSD licensed)
  • most of the business logic
  • distributed via a disk image so you can upload it to the cloud
  • semi-algorithmic trading. The idea is to fill the niche between something totally algo and something non-algo. So what you have is a set of prepackaged scripts that you can run when see a trade opportunity. or if you see something that you want to run for an hour while are looking at the screen.
  • works with bitcoin and Shanghai equity options

Technologies

What is unique about bittrader?
  • 100% open source / BSD licensed / all development on github
  • thin javascript client
  • distributed as a virtual disk image. Upload the image to a cloud or run as a virtual machine with VirtualBox.

Bittrader pulls together several different technologies into one system

  • opengamma - backend engine
  • jqueryui (and friends) - display
  • flotr2 - plotting
  • mageia - infrastructure
  • zipline or pyalgotrade - Backtesting
  • rNotebook - data exploration/reports
  • ipython notebooks
  • quantlib - algo trading
